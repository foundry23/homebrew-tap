# Foundry23 Tap

## How do I install these formulae?

`brew install foundry23/tap/<formula>`

Or `brew tap foundry23/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "foundry23/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
